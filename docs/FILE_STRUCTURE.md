# File structure

## Top-level layout

- [AGENTS.md](../AGENTS.md): Agent workflow and environment rules.
- [README.md](../README.md): Project overview and quick start.
- [mkdocs.yml](../mkdocs.yml): MkDocs configuration and site navigation.
- [generate_pages.py](../generate_pages.py): Page-generation entry point for
  subject indexes, topic pages, the generated nav block, and the self-test
  manifest.
- [run_bbq_tasks.py](../run_bbq_tasks.py): BBQ batch task runner (imports
  `bioproblems_site`).
- [topics_metadata.yml](../topics_metadata.yml): Single source of truth for subjects
  and topics (titles, descriptions, aliases, links).
- [bioproblems_site/](../bioproblems_site/): Importable generator package for
  metadata, scanning, rendering, nav updates, and self-test manifest creation.
- [site_docs/](../site_docs/): MkDocs content root for pages and assets.
- `site/`: Built site output (generated, git ignored).
- [docs/](.): Repo documentation and standards.
- [bbq_control/](../bbq_control/): BBQ task CSVs, settings, and runner scripts.
- [tools/](../tools/): Helper scripts for content generation tasks.
- [devel/](../devel/): Developer utilities (versioning, changelog tooling).
- [tests/](../tests/): Pytest suite, lint gates, and ASCII compliance checks.

## Key subtrees

- [bioproblems_site/](../bioproblems_site/): Generation package, including
  [bioproblems_site/pipeline.py](../bioproblems_site/pipeline.py),
  [bioproblems_site/metadata.py](../bioproblems_site/metadata.py),
  [bioproblems_site/topic_aliases.py](../bioproblems_site/topic_aliases.py), and
  [bioproblems_site/selftest_manifest.py](../bioproblems_site/selftest_manifest.py).
- [site_docs/](../site_docs/): One folder per subject (for example
  [site_docs/biochemistry/](../site_docs/biochemistry/),
  [site_docs/genetics/](../site_docs/genetics/),
  [site_docs/molecular_biology/](../site_docs/molecular_biology/)), each holding
  `topicNN/index.md` pages.
- [site_docs/assets/](../site_docs/assets/): Shared CSS, JS, fonts, images, and
  generated data.
- [site_docs/assets/scripts/selftest_progress.js](../site_docs/assets/scripts/selftest_progress.js): Browser self-test completion tracking.
- [site_docs/progress/](../site_docs/progress/): Self-test progress dashboard page.
- [site_docs/daily_puzzles/](../site_docs/daily_puzzles/): Daily puzzle pages and
  scripts.
- [site_docs/tutorials/](../site_docs/tutorials/): BBQ and LMS import tutorials.
- [bbq_control/task_files/](../bbq_control/task_files/): Per-subject task CSVs (for
  example `biochem_tasks1.csv`, `biotech_tasks.csv`, `other_tasks.csv`).

## Generated artifacts

Git-ignored outputs and local-only files are listed in [.gitignore](../.gitignore),
including the built `site/` directory, `report_*.txt` lint reports,
`bbq_generation.log*` files, and the `bbq_converter.py` symlink. Subject index
pages and topic pages under [site_docs/](../site_docs/) are regenerated by
[generate_pages.py](../generate_pages.py); subject and topic folders with no
questions are omitted from the rendered nav.

## Where to add new work

- New site content: [site_docs/](../site_docs/).
- New subject or topic: [topics_metadata.yml](../topics_metadata.yml) plus a nav label
  in [mkdocs.yml](../mkdocs.yml).
- New generation logic: a module under [bioproblems_site/](../bioproblems_site/).
- New BBQ tasks: a per-subject CSV in
  [bbq_control/task_files/](../bbq_control/task_files/).
- New helper scripts: [tools/](../tools/).
- New docs: [docs/](.).
- New tests: [tests/](../tests/).
